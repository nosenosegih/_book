はじめに

y.u mobile向けヒューマンコア在庫管理 Web API

GraphQL による Web API 機能を実現します。

この記事は GraphQL についての説明はしていませんので、GraphQL 自体の仕様についてGraphQL公式サイト等でご確認ください。

認証

このAPIはOAuth 2.0を利用して認証を行います。APIリクエストを行うには、有効なアクセストークンが必要です。

トークンの取得

アクセストークンは、ヒューマンコアが提供します。トークンを取得するための手順は以下の通りです：

  1. ヒューマンコアにAPIの利用申請を行ってください。
  2. 申請が承認された場合、アクセストークンが提供されます。
  3. このトークンはAPIリクエストのヘッダーに含める必要があります。
トークンの使用

トークンは、APIリクエストのヘッダーにAuthorizationフィールドとして含めてください。

Authorization: Bearer [アクセストークン]

エンドポイント

エンドポイントはQuery/Mutationで共通です。

api endpoint: https://shop.humancore.co.jp/yu-shop/api

商品画像URL

商品画像のURLを取得するには、以下の「商品画像ベースURL」とAPIレスポンスの「画像ファイル名」を組み合わせてください。

商品画像ベースURL: https://shop.humancore.co.jp/yu-shop/upload/save_image

例: 画像ファイル名が product123.jpg の場合、フルURLは [https://shop.humancore.co.jp/yu-shop/upload/save_image/product123.jpg] となります。
