商品在庫の更新

商品在庫の更新

Mutation

mutation {
  api42ExtendedUpdateStock(code: "sp-001", quantity: 2) {
    code
    stock
  }
}

入力フィールド

  • code (String!): 更新する商品のコード
  • quantity (Int!): 在庫を減少させる数量

返り値

  • code (String): 更新された商品のコード
  • stock (Int): 更新後の商品の在庫数

実行結果

{
  "data": {
    "api42ExtendedUpdateStock": {
      "code": "sp-001",
      "stock": 18
    }
  }
}
